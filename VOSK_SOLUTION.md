# ✅ РЕШЕНИЕ: Vosk - офлайн распознавание речи

## Что это?

**Vosk** - бесплатная библиотека распознавания речи которая работает **полностью в браузере** без серверов.

## Преимущества:

✅ **100% бесплатно** - никаких API ключей
✅ **Работает офлайн** - не нужен интернет после загрузки
✅ **Обходит Web Speech API** - использует Web Audio API напрямую
✅ **Нет проблем с разрешениями** - микрофон запрашивается один раз
✅ **Работает на всех устройствах** - Android, iOS, десктоп

## Как работает:

1. **На десктопе**: используется Web Speech API (работает отлично)
2. **На мобильных**: автоматически переключается на Vosk

## Использование на телефоне:

1. Открой https://chess-voice-web.vercel.app
2. Нажми "Начать партию"
3. Нажми "Включить прослушивание"
4. При первом запуске увидишь "⏳ Загружаю модель распознавания..." (5-10 секунд)
5. После загрузки: "🎤 Слушаю..."
6. Говори ходы!

## Размер:

Модель Vosk весит ~40 МБ, но загружается один раз и кэшируется браузером.

## Если не работает:

1. Проверь что дал разрешение на микрофон
2. Подожди пока загрузится модель (первый раз 5-10 сек)
3. Проверь консоль - должно быть `[Vosk] Модель загружена`

## Это финальное решение!

Vosk обходит ВСЕ ограничения Web Speech API на мобильных и работает **100% бесплатно** без серверов.
