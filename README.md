# Цифровой Суфлёр - Веб-версия

Голосовой шахматный помощник с поддержкой Web Speech API.

## Возможности

- ✅ Голосовой ввод ходов (работает в Chrome/Edge)
- ✅ Озвучка ходов компьютера
- ✅ Stockfish.js для игры против компьютера
- ✅ Команды "отмена" и "ещё раз"
- ✅ Адаптивный дизайн для мобильных устройств

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## Деплой на Vercel

### Вариант 1: Через GitHub (рекомендуется)

1. Создай новый репозиторий на GitHub
2. Запуши код:
```bash
git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_USERNAME/chess-voice-web.git
git push -u origin main
```

3. Зайди на https://vercel.com
4. Нажми "Add New" → "Project"
5. Импортируй свой GitHub репозиторий
6. Vercel автоматически определит настройки (Vite)
7. Нажми "Deploy"

### Вариант 2: Через Vercel CLI

1. Установи Vercel CLI:
```bash
npm install -g vercel
```

2. Залогинься:
```bash
vercel login
```

3. Задеплой:
```bash
vercel
```

4. Следуй инструкциям в терминале

## Использование

1. Выбери цвет (белые/чёрные)
2. Нажми "Начать игру"
3. Нажми "🎤 Сказать ход"
4. Скажи ход, например: "e2 e4"
5. Компьютер ответит своим ходом

### Голосовые команды

- **"e2 e4"** - сделать ход
- **"отмена"** - отменить последний ход
- **"ещё раз"** - повторить озвучку последнего хода

## Требования браузера

- Chrome/Edge (рекомендуется) - полная поддержка Web Speech API
- Firefox - частичная поддержка
- Safari - ограниченная поддержка

## Примечания

- Для голосового ввода требуется HTTPS (на Vercel автоматически)
- Первый запуск может быть медленным из-за загрузки Stockfish.js
- Микрофон требует разрешения браузера
