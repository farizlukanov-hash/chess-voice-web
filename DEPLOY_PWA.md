# Как задеплоить PWA версию на Vercel

## Что сделано:

✅ Добавлен PWA манифест (`manifest.json`)
✅ Создан Service Worker (`sw.js`)
✅ Добавлены иконки приложения
✅ Кнопка "Установить приложение" в интерфейсе
✅ Мета-теги для iOS и Android

## Деплой на Vercel:

### Вариант 1: Через GitHub Desktop (самый простой)

1. Открой GitHub Desktop
2. Выбери репозиторий `chess-voice-web`
3. Нажми "Push origin"
4. Vercel автоматически задеплоит через 1-2 минуты

### Вариант 2: Через браузер

1. Открой https://github.com/farizlukanov-hash/chess-voice-web
2. Нажми "Add file" → "Upload files"
3. Перетащи все файлы из папки `C:\Users\kirar\OneDrive\Desktop\bat\chess-voice-web`
4. Нажми "Commit changes"
5. Vercel автоматически задеплоит

### Вариант 3: Через командную строку (если есть доступ)

```bash
cd C:\Users\kirar\OneDrive\Desktop\bat\chess-voice-web
git push
```

## Как использовать PWA на телефоне:

### Android (Chrome):

1. Открой https://chess-voice-web.vercel.app
2. Внизу появится кнопка "📱 Установить приложение"
3. Нажми на неё
4. Подтверди установку
5. Иконка появится на домашнем экране
6. Открывай приложение с домашнего экрана (не через браузер!)

### iOS (Safari):

1. Открой https://chess-voice-web.vercel.app
2. Нажми кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Выбери "На экран «Домой»"
4. Нажми "Добавить"
5. Открывай приложение с домашнего экрана

## Почему PWA решает проблему с микрофоном:

✅ **Больше прав**: PWA получает те же права что и нативные приложения
✅ **Стабильное разрешение**: Микрофон запрашивается один раз при установке
✅ **Нет ограничений браузера**: Работает как отдельное приложение
✅ **Офлайн режим**: Service Worker кэширует файлы
✅ **Полноэкранный режим**: Убирает адресную строку браузера

## Проверка что PWA работает:

После установки на домашний экран:
- Иконка должна быть видна на главном экране
- При открытии НЕ должно быть адресной строки браузера
- В консоли должно быть: `[PWA] Приложение уже установлено`

## Если всё равно не работает:

Тогда единственный вариант - **нативное Android приложение** (APK который я уже собрал).

Но сначала попробуй PWA - в 90% случаев это решает проблему с микрофоном на мобильных!
